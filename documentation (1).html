Getting Started

Welcome to iProyals! This guide will help you get up and running with our proxy services in minutes.

1. Create Your Account

Sign up for an iProyals account and choose the proxy type that best fits your needs.

2. Get Your Credentials

Navigate to your dashboard to find your authentication credentials.

3. Configure Your Connection

Use the connection details provided in your dashboard to configure your application.

Connection Methods

HTTP/HTTPS Proxies

Standard proxy format for web browsers and HTTP clients:

Proxy Host: proxy.iproyals.com
Proxy Port: 8080
Username: your_username
Password: your_password

SOCKS5 Proxies

For applications requiring SOCKS5 protocol support:

Proxy Host: socks.iproyals.com
Proxy Port: 1080
Username: your_username
Password: your_password

API Reference

Authentication

All API requests require authentication using your API key:

Authorization: Bearer YOUR_API_KEY

Get Account Info

GET https://api.iproyals.com/v1/account

Response:
{
  "username": "your_username",
  "balance": 100.00,
  "proxy_count": 10,
  "bandwidth_used": "50GB"
}

List Active Proxies

GET https://api.iproyals.com/v1/proxies

Response:
{
  "proxies": [
    {
      "id": "proxy_123",
      "type": "residential",
      "ip": "192.168.1.1",
      "port": 8080,
      "country": "US"
    }
  ]
}

Code Examples

Python Example

import requests

proxies = {
    'http': 'http://username:password@proxy.iproyals.com:8080',
    'https': 'http://username:password@proxy.iproyals.com:8080'
}

response = requests.get('https://api.ipify.org', proxies=proxies)
print(f'Your IP: {response.text}')

Node.js Example

const axios = require('axios');

const proxy = {
  host: 'proxy.iproyals.com',
  port: 8080,
  auth: {
    username: 'your_username',
    password: 'your_password'
  }
};

axios.get('https://api.ipify.org', { proxy })
  .then(response => console.log('Your IP:', response.data));

cURL Example

curl -x proxy.iproyals.com:8080 \
     -U username:password \
     https://api.ipify.org
