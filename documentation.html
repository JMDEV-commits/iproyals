Getting Started

Welcome to iProyals! This guide will help you get up and running with our proxy services in minutes.

1. Create Your Account

Sign up for an iProyals account and choose the proxy type that best fits your needs. You'll receive instant access to our dashboard.

2. Get Your Credentials

Navigate to your dashboard to find your authentication credentials. You'll need these to connect to our proxy network.

3. Configure Your Connection

Use the connection details provided in your dashboard to configure your application or browser.

Connection Methods

HTTP/HTTPS Proxies

Standard proxy format for web browsers and HTTP clients:

Proxy Host: proxy.iproyals.com
Proxy Port: 8080
Username: your_username
Password: your_password

SOCKS5 Proxies

For applications requiring SOCKS5 protocol support:

Proxy Host: socks.iproyals.com
Proxy Port: 1080
Username: your_username
Password: your_password

API Reference

Authentication

All API requests require authentication using your API key. Include your API key in the request header:

Authorization: Bearer YOUR_API_KEY

Get Account Info

Retrieve your account information and usage statistics.

GET https://api.iproyals.com/v1/account

Response:
{
  "username": "your_username",
  "balance": 100.00,
  "proxy_count": 10,
  "bandwidth_used": "50GB",
  "bandwidth_limit": "100GB"
}

List Active Proxies

Get a list of all your active proxy connections.

GET https://api.iproyals.com/v1/proxies

Response:
{
  "proxies": [
    {
      "id": "proxy_123",
      "type": "residential",
      "ip": "192.168.1.1",
      "port": 8080,
      "country": "US",
      "status": "active"
    }
  ]
}

Generate Proxy List

Generate a custom list of proxy endpoints.

POST https://api.iproyals.com/v1/proxies/generate

Request Body:
{
  "type": "residential",
  "country": "US",
  "count": 10,
  "protocol": "http"
}

Response:
{
  "proxies": [
    "192.168.1.1:8080:username:password",
    "192.168.1.2:8080:username:password"
  ]
}

Code Examples

Python Example

import requests

proxies = {
    'http': 'http://username:password@proxy.iproyals.com:8080',
    'https': 'http://username:password@proxy.iproyals.com:8080'
}

response = requests.get('https://api.ipify.org', proxies=proxies)
print(f'Your IP: {response.text}')

Node.js Example

const axios = require('axios');

const proxy = {
  host: 'proxy.iproyals.com',
  port: 8080,
  auth: {
    username: 'your_username',
    password: 'your_password'
  }
};

axios.get('https://api.ipify.org', { proxy })
  .then(response => console.log('Your IP:', response.data))
  .catch(error => console.error('Error:', error));

cURL Example

curl -x proxy.iproyals.com:8080 \
     -U username:password \
     https://api.ipify.org

Best Practices

Connection Pool Management

Reuse connections when possible to improve performance and reduce overhead. Maintain a pool of active connections for high-volume operations.

Error Handling

Implement proper retry logic for failed requests. Use exponential backoff to avoid overwhelming the proxy network during outages.

Rate Limiting

Respect rate limits to ensure optimal performance. Distribute requests across multiple proxies when performing high-volume operations.

Session Management

Use sticky sessions when you need to maintain the same IP address across multiple requests. This is crucial for activities like account management.

Troubleshooting

Authentication Errors

Verify that your username and password are correct. Ensure there are no extra spaces or special characters that might be causing issues.

Connection Timeouts

Check your network connectivity and firewall settings. Ensure that your application is configured to use the correct proxy host and port.

Slow Performance

Try using datacenter proxies instead of residential for speed-critical tasks. Check your bandwidth usage and upgrade your plan if necessary.
