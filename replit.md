# iProyals - Static Website

## Project Overview
A professional static website for iProyals, a provider of premium proxy and VPN services. The site is modern, responsive, and SEO-optimized with multiple pages.

## Tech Stack
- **HTML5** - Semantic markup
- **CSS3** - Custom styling with Grid, Flexbox, and CSS Variables
- **JavaScript (ES6+)** - Vanilla JS for interactivity
- **Fonts** - Google Fonts (Sora and DM Sans)

## Project Structure
```
/
├── index.html          # Homepage (placeholder)
├── about.html          # Company info
├── contact.html        # Contact form
├── documentation.html  # API reference
├── pricing.html        # Pricing tables
├── products.html       # Products overview
├── use-cases.html      # Industry use cases
├── styles.css          # Global styles
├── main.js             # Client-side JS
└── README.md           # Project documentation
```

## Running the Project
- **Workflow**: "Start application" runs `python3 -m http.server 5000 --bind 0.0.0.0`
- **Port**: 5000
- **Type**: Static file server

## Deployment
- **Target**: Static
- **Public Directory**: `.` (root)
- No build step required
